# gcs

[谷歌云存储](https://cloud.google.com/storage/)

## 配置

### 服务

| 名称   | 必填 | 评论               |
| ---- | -- | ---------------- |
| `凭据` | Y  | 仅支持 `api_key` 协议 |

### 存储器

| 名称     | 必填 | 评论       |
| ------ | -- | -------- |
| `名称`   | Y  | 桶名称      |
| `工作目录` | N  | 工作目录     |
| `项目ID` | Y  | 谷歌云项目 id |
